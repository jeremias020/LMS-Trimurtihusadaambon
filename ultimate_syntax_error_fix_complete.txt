## ✅ **ULTIMATE SOLUTION - Separation of Concerns:**

### **🔧 Final Fix Applied:**
```php
// BEFORE (Complex inline - causes parser confusion)
const assessmentCriteria = @json(isset($allCriteria) && $allCriteria->count() > 0 ? 
    $allCriteria->map(function($item, $key) {
        return [
            'id' => $item->id ?? $key,
            'nama_kriteria' => $item->name ?? 'Kriteria ' . ($key + 1),
            'deskripsi' => $item->description ?? 'Deskripsi kriteria',
            'bobot' => ($item->weight ?? 5) / 100,
            'category' => 'umum'
        ];
    })->groupBy('category')->toArray() : []
);

// AFTER (Clean separation - parser friendly)
@if(isset($allCriteria) && $allCriteria->count() > 0)
    @php
        $criteriaData = $allCriteria->map(function($item, $key) {
            return [
                'id' => $item->id ?? $key,
                'nama_kriteria' => $item->name ?? 'Kriteria ' . ($key + 1),
                'deskripsi' => $item->description ?? 'Deskripsi kriteria',
                'bobot' => ($item->weight ?? 5) / 100,
                'category' => 'umum'
            ];
        })->groupBy('category')->toArray();
    @endphp
    const assessmentCriteria = {!! json_encode($criteriaData) !!};
@else
    const assessmentCriteria = [];
@endif
```
