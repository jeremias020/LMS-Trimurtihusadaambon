=== PERBAIKAN ROUTE ERROR ABSENSI GURU ===

🔍 PROBLEM IDENTIFIED:
- Error: Route [absensi.create] not defined
- View menggunakan route tanpa prefix 'guru.'
- Route sebenarnya ada di dalam prefix 'guru'
- Route names: guru.absensi.index, guru.absensi.create, dll

✅ ROOT CAUSE:
Route absensi berada di dalam group:
```php
Route::prefix('guru')->name('guru.')->middleware(['auth', 'guru'])->group(function () {
    Route::resource('absensi', GuruAttendanceController::class)->names([
        'index' => 'absensi.index',
        'create' => 'absensi.create', 
        // ...
    ]);
});
```

Jadi route yang sebenarnya adalah:
- guru.absensi.index
- guru.absensi.create
- guru.absensi.store
- guru.absensi.edit
- dll

✅ FIXES APPLIED:

1. INDEX VIEW (guru/absensi/index.blade.php):
   - route('absensi.create') → route('guru.absensi.create')
   - route('absensi.edit') → route('guru.absensi.edit')

2. CREATE VIEW (guru/absensi/create.blade.php):
   - route('absensi.index') → route('guru.absensi.index') (breadcrumb)
   - route('absensi.index') → route('guru.absensi.index') (page-actions)
   - route('absensi.store') → route('guru.absensi.store') (form action)
