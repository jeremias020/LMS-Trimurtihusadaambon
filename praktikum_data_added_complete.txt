## ✅ **Technical Details:**

### **🎯 1. Database Schema:**
```sql
-- Tabel practicals
CREATE TABLE practicals (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    guru_id BIGINT NOT NULL,
    subject_id BIGINT NOT NULL,
    judul VARCHAR(255) NOT NULL,
    deskripsi TEXT,
    tanggal DATE,
    waktu_mulai TIME,
    waktu_selesai TIME,
    lokasi VARCHAR(255),
    durasi INT,
    skill_level VARCHAR(50),
    tools TEXT,
    bahan TEXT,
    instruksi TEXT,
    keselamatan TEXT,
    kelas_id BIGINT,
    max_score INT DEFAULT 100,
    is_published BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    deleted_at TIMESTAMP NULL
);
```

### **🎯 2. Model Relationships:**
```php
// Practical Model
class Practical extends Model
{
    public function guru()
    {
        return $this->belongsTo(User::class, 'guru_id');
    }
    
    public function subject()
    {
        return $this->belongsTo(Subject::class, 'subject_id');
    }
    
    public function kelas()
    {
        return $this->belongsTo(Kelas::class, 'kelas_id');
    }
}
```

### **🎯 3. Controller Logic:**
```php
// Get practicals for current guru
$practicals = Practical::where('guru_id', $guruId)
    ->where('is_published', true)
    ->with('subject')
    ->latest()
    ->get();

// Pass to view
return view('guru.penilaian.auto_with_criteria', compact(
    'subjects', 'classes', 'students', 'practicals'
));
```
