1. CSS ENHANCEMENT:

   ✅ Gunakan font-size yang konsisten (14px)
   ✅ Gunakan font-family yang sama (Arial, sans-serif)
   ✅ Tambahkan min-height untuk konsistensi visual
   ✅ Gunakan flexbox untuk proper alignment

2. SELECTOR OPTIMIZATION:
   ✅ Multiple approaches untuk cross-browser compatibility:
     - General .page-link selector
     - :contains() pseudo-selector
     - Attribute selectors untuk aria-label
   ✅ !important rules untuk override default styles
   ✅ Line-height optimization untuk proper vertical alignment

📱 TECHNICAL IMPLEMENTATION:

CSS RULES APPLIED:
```css
/* Force pagination symbols to be small and consistent */
.pagination .page-link {
    font-size: 14px !important;
    font-family: Arial, sans-serif !important;
    line-height: 1.2 !important;
}

.pagination .page-link:empty {
    font-size: 14px !important;
    line-height: 1.2 !important;
}

/* Specific rules for arrow symbols - ensure consistent size */
.pagination .page-link {
    font-size: 14px !important;
    min-height: 38px !important;
    display: flex !important;
    align-items: center !important;
    justify-content: center !important;
}

/* Force arrow symbols to be consistent */
.page-link:contains("‹"),
.page-link:contains("›") {
    font-size: 14px !important;
    font-family: Arial, sans-serif !important;
    line-height: 1 !important;
}

/* Alternative approach using attribute selectors */
.page-link[aria-label*="previous"],
.page-link[aria-label*="next"] {
    font-size: 14px !important;
    font-family: Arial, sans-serif !important;
    line-height: 1.2 !important;
}
```

PAGINATION HTML STRUCTURE:
```blade
<!-- Previous Arrow -->
<span class="page-link">‹</span>

<!-- Next Arrow -->
<span class="page-link">›</span>
```
