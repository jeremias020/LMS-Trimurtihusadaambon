=== PERBAIKAN HTML ENTITY ISSUE MANAJEMEN ABSENSI ===

🔍 PROBLEM IDENTIFIED:
- Icon < dan > yang sangat besar muncul di nama siswa
- HTML entity tidak di-escape dengan benar
- Karakter khusus dalam data siswa menyebabkan rendering error
- Kemungkinan data mengandung < atau > karakter

✅ FIXES APPLIED:

1. HTML ENTITY ESCAPING:
   - SEBELUM: {{ $attendance->siswa?->name }}
   - SESUDAH: {{ e($attendance->siswa?->name) }}
   - Laravel e() helper untuk proper HTML escaping

2. FIXED FIELDS:
   - Nama siswa: {{ e($attendance->siswa?->name ?? 'N/A') }}
   - NIS: {{ e($attendance->siswa?->siswa?->nis ?? '-') }}
   - Kelas: {{ e($attendance->siswa?->kelas?->name ?? 'Tidak diketahui') }}
   - Keterangan: {{ e($attendance->keterangan ?? '-') }}
   - Alt text avatar: {{ e($attendance->siswa?->name ?? 'Siswa') }}

3. SECURITY IMPROVEMENT:
   - Prevent XSS attacks
   - Proper HTML entity conversion
   - Safe data rendering
   - Consistent escaping across all user data
