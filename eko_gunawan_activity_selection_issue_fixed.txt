2. TECHNICAL ISSUES:

   ❌ Tidak ada event listener untuk perubahan siswa
   ❌ Empty state handling tidak spesifik per siswa
   ❌ Debug information tidak tersedia untuk troubleshooting

2. STUDENT SELECTION HANDLER:
```javascript
function onStudentChange() {
    const selectedStudent = document.getElementById('siswa_id')?.value;
    const typeSelect = document.getElementById('type');
    
    if (selectedStudent) {
        // Reset type selection
        typeSelect.value = '';
        toggleAssessmentType();
        
        // Show info about selected student
        const studentOption = document.querySelector(`#siswa_id option[value="${selectedStudent}"]`);
        if (studentOption) {
            const studentName = studentOption.textContent;
            showNotification(`Siswa dipilih: ${studentName}. Silakan pilih jenis penilaian.`, 'success');
        }
    }
}
```
